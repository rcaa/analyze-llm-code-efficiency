ID: 2432
Title: The Employee That Worked on the Longest Task
URL: https://leetcode.com/problems/the-employee-that-worked-on-the-longest-task/
Status: OK

solution: 
```cpp
int findLongestTask(int n, vector<vector<int>> logs) {
    int previous_time = 0;
    int max_duration = 0;
    int result_id = -1;
    
    for (const auto& log : logs) {
        int current_id = log[0];
        int current_leave = log[1];
        int duration = current_leave - previous_time;
        
        if (duration > max_duration) {
            max_duration = duration;
            result_id = current_id;
        } else if (duration == max_duration) {
            if (current_id < result_id) {
                result_id = current_id;
            }
        }
        
        previous_time = current_leave;
    }
    
    return result_id;
}
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(1)
energy implications: LOW
explanation: The solution efficiently processes each task exactly once, calculating the duration and updating the maximum duration and corresponding employee id as needed. This approach ensures optimal performance with minimal resource usage.
