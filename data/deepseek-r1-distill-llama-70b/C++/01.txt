Title: Online Majority Element In Subarray
URL: https://leetcode.com/problems/online-majority-element-in-subarray/
Status: OK

solution: Implement a segment tree where each node stores a map of element counts in its range. For each query, collect the counts from the relevant nodes and check for the majority element.

efficiency: MEDIUM

time complexity: O(n log n) for building the tree, O((log n)^2) per query

space complexity: O(n log n)

energy implications: MEDIUM

explanation: The solution uses a segment tree to efficiently handle range queries. Each node stores a map of element counts, allowing quick accumulation of counts for any subarray. The majority element is identified by checking if any element's count meets the threshold. This approach balances time and space complexity, making it suitable for the given constraints.
