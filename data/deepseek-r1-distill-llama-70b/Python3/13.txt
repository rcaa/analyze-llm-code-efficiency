ID: 1207
Title: Unique Number of Occurrences
URL: https://leetcode.com/problems/unique-number-of-occurrences/
Status: OK

solution: 
```python
from collections import Counter

def unique_occurrences(arr):
    counts = Counter(arr)
    return len(counts.values()) == len(set(counts.values()))
```

efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: This solution efficiently counts occurrences using a dictionary, then checks for uniqueness by comparing list and set lengths, ensuring optimal performance.
