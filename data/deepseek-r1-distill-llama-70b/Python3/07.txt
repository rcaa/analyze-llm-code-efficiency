To solve this problem, we need to find the lexicographically smallest permutation of an array of positive integers such that the concatenated result is divisible by a given integer \( k \). If no such permutation exists, we return an empty list.

### Approach
The approach involves using dynamic programming (DP) combined with a breadth-first search (BFS) to efficiently explore all possible permutations while keeping track of the smallest lexicographical order. Here's a detailed breakdown:

1. **Precomputation**: 
   - Compute the length of each number in the array.
   - Compute each number modulo \( k \).
   - Precompute powers of 10 modulo \( k \) up to the maximum length of the numbers in the array. This helps in efficiently computing remainders when concatenating numbers.

2. **Dynamic Programming Initialization**:
   - Initialize a DP dictionary where each key is a tuple (mask, remainder) and the value is the smallest permutation leading to that state. The mask is a bitmask representing which numbers have been used, and the remainder is the result of the concatenated number modulo \( k \).

3. **Breadth-First Search (BFS)**:
   - Use a queue to process each state in BFS order. For each state, try adding each unused number in ascending order to ensure the smallest lexicographical permutation is found first.
   - For each transition, compute the new remainder and new mask. Update the DP dictionary if the new permutation is smaller than the existing one.

4. **Check for Solution**:
   - After processing all states, check if the final state (all numbers used and remainder 0) exists. If it does, return the corresponding permutation; otherwise, return an empty list.

### Solution Code
```python
def find_divisible_concatenation(nums, k):
    n = len(nums)
    if n == 0:
        return []
    
    len_nums = [len(str(num)) for num in nums]
    mod_nums = [num % k for num in nums]
    
    max_len = max(len_nums)
    pow10 = [1]  # pow10[0] = 1
    for i in range(1, max_len + 1):
        pow10.append((pow10[-1] * 10) % k)
    
    dp = {}
    for i in range(n):
        mask = 1 << i
        rem = mod_nums[i]
        key = (mask, rem)
        dp[key] = [nums[i]]
    
    from collections import deque
    queue = deque()
    for i in range(n):
        mask = 1 << i
        rem = mod_nums[i]
        queue.append((mask, rem, [nums[i]]))
    
    while queue:
        mask, rem, perm = queue.popleft()
        if bin(mask).count('1') == n:
            if rem == 0:
                return perm
            else:
                continue
        
        next_indices = [j for j in range(n) if not (mask & (1 << j))]
        next_indices.sort(key=lambda x: nums[x])
        
        for j in next_indices:
            new_rem = (rem * pow10[len_nums[j]] + mod_nums[j]) % k
            new_mask = mask | (1 << j)
            new_perm = perm + [nums[j]]
            key = (new_mask, new_rem)
            
            if key not in dp or new_perm < dp[key]:
                dp[key] = new_perm
                queue.append((new_mask, new_rem, new_perm))
    
    return []
```

### Explanation
- **Precomputation**: This step ensures that we have all necessary values precomputed to efficiently calculate remainders during concatenation.
- **DP Initialization**: Each number starts as a potential beginning of the permutation, initializing the DP with these starting points.
- **BFS Processing**: By processing each state and expanding it by adding unused numbers in ascending order, we ensure that the smallest lexicographical permutation is found first.
- **Efficiency**: The approach efficiently prunes the search space using bitmasking and remainder tracking, ensuring that we do not explore unnecessary permutations.

This method efficiently finds the smallest permutation that meets the divisibility condition, leveraging dynamic programming and BFS to manage the state space effectively.
