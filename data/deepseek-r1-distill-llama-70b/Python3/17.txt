Title: Most Frequent Even Element
URL: https://leetcode.com/problems/most-frequent-even-element/
Status: OK

solution: |
    def most_frequent_even(nums):
        count = {}
        for num in nums:
            if num % 2 == 0:
                count[num] = count.get(num, 0) + 1
        if not count:
            return -1
        max_freq = max(count.values())
        candidates = [k for k, v in count.items() if v == max_freq]
        return min(candidates)

efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: The solution efficiently counts even elements using a dictionary, then finds the most frequent even element, resolving ties by selecting the smallest. This approach ensures optimal performance with linear time complexity.
