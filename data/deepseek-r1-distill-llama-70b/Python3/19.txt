Title: The Employee That Worked on the Longest Task
URL: https://leetcode.com/problems/the-employee-that-worked-on-the-longest-task/
Status: OK

solution: |
    def findLongestTask(n, logs):
        max_duration = 0
        result_id = -1
        for i in range(len(logs)):
            if i == 0:
                duration = logs[i][1]
            else:
                duration = logs[i][1] - logs[i-1][1]
            if duration > max_duration:
                max_duration = duration
                result_id = logs[i][0]
            elif duration == max_duration:
                if logs[i][0] < result_id:
                    result_id = logs[i][0]
        return result_id
efficiency: HIGH
time complexity: O(n)
space complexity: O(1)
energy implications: LOW
explanation: The solution iterates through each task once, calculating the duration by comparing consecutive tasks. It keeps track of the maximum duration and the corresponding employee id, updating as necessary. This approach ensures optimal performance with linear time complexity and constant space usage.
