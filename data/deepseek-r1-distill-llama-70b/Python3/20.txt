Title: Largest Positive Integer That Exists With Its Negative
URL: https://leetcode.com/problems/largest-positive-integer-that-exists-with-its-negative/
Status: OK

solution: 
```python
def largest_k(nums):
    s = set(nums)
    max_k = -1
    for num in nums:
        if num > 0 and -num in s:
            if num > max_k:
                max_k = num
    return max_k if max_k != -1 else -1
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: The solution uses a set for O(1) lookups to efficiently check for the existence of the negative counterpart of each positive number. It iterates through the array once, making it linear in time complexity. The space complexity is linear due to the set storage. This approach ensures we find the largest k efficiently.
