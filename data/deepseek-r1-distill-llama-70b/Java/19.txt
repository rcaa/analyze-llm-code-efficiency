ID: 2432
Title: The Employee That Worked on the Longest Task
URL: https://leetcode.com/problems/the-employee-that-worked-on-the-longest-task/
Status: OK

solution: 
```java
public class Solution {
    public int findLongestTask(int n, int[][] logs) {
        int prevTime = 0;
        int maxDuration = 0;
        int resultId = -1;
        
        for (int[] log : logs) {
            int currentTime = log[1];
            int duration = currentTime - prevTime;
            
            if (duration > maxDuration) {
                maxDuration = duration;
                resultId = log[0];
            } else if (duration == maxDuration) {
                if (log[0] < resultId) {
                    resultId = log[0];
                }
            }
            
            prevTime = currentTime;
        }
        
        return resultId;
    }
}
```
efficiency: HIGH
time complexity: O(m) where m is the number of logs
space complexity: O(1)
energy implications: LOW
explanation: The solution efficiently processes each log entry exactly once, calculating the duration of each task and tracking the maximum duration and corresponding employee id. This approach ensures optimal performance with minimal energy usage.
