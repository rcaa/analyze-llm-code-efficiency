Title: Unique Number of Occurrences
URL: https://leetcode.com/problems/unique-number-of-occurrences/
Status: OK

solution: 
```java
public boolean uniqueOccurrences(int[] arr) {
    HashMap<Integer, Integer> frequency = new HashMap<>();
    for (int num : arr) {
        frequency.put(num, frequency.getOrDefault(num, 0) + 1);
    }
    HashSet<Integer> counts = new HashSet<>(frequency.values());
    return counts.size() == frequency.size();
}
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: The solution uses a HashMap to count occurrences and a HashSet to track unique counts, ensuring efficient checking of unique occurrence counts with linear time complexity.
