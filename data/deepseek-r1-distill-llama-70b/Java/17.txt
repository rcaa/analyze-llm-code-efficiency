ID: 2404
Title: Most Frequent Even Element
URL: https://leetcode.com/problems/most-frequent-even-element/
Status: OK

solution: 
```java
import java.util.HashMap;
import java.util.Map;

public class Main {
    public static int mostFrequentEven(int[] nums) {
        Map<Integer, Integer> evenCount = new HashMap<>();
        for (int num : nums) {
            if (num % 2 == 0) {
                evenCount.put(num, evenCount.getOrDefault(num, 0) + 1);
            }
        }
        if (evenCount.isEmpty()) {
            return -1;
        }
        int maxFreq = 0;
        for (int count : evenCount.values()) {
            if (count > maxFreq) {
                maxFreq = count;
            }
        }
        int minNum = Integer.MAX_VALUE;
        for (Map.Entry<Integer, Integer> entry : evenCount.entrySet()) {
            if (entry.getValue() == maxFreq && entry.getKey() < minNum) {
                minNum = entry.getKey();
            }
        }
        return minNum;
    }
}
```

efficiency: HIGH

time complexity: O(n)

space complexity: O(n)

energy implications: LOW

explanation: The solution efficiently counts occurrences of even numbers using a HashMap, then determines the most frequent and smallest in case of ties, ensuring optimal performance with linear time complexity.
