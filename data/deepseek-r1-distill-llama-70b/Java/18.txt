ID: 2418
Title: Sort the People
URL: https://leetcode.com/problems/sort-the-people/
Status: OK

solution: 
```java
class Person {
    String name;
    int height;
    
    public Person(String name, int height) {
        this.name = name;
        this.height = height;
    }
}

public class Main {
    public static String[] sortNamesByHeight(String[] names, int[] heights) {
        List<Person> people = new ArrayList<>();
        for (int i = 0; i < names.length; i++) {
            people.add(new Person(names[i], heights[i]));
        }
        
        Collections.sort(people, (a, b) -> Integer.compare(b.height, a.height));
        
        String[] result = new String[names.length];
        for (int i = 0; i < result.length; i++) {
            result[i] = people.get(i).name;
        }
        return result;
    }
}
```

efficiency: HIGH
time complexity: O(n log n)
space complexity: O(n)
energy implications: LOW
explanation: This solution pairs each name with its height in a custom object, sorts the list of objects by height in descending order using an efficient comparator, and then extracts the sorted names. The use of a custom class and sorting algorithm ensures clarity and efficiency.
