Title: Online Majority Element In Subarray
URL: https://leetcode.com/problems/online-majority-element-in-subarray/
Status: OK

solution: Implement a MajorityChecker class using a hashmap to store each element's indices. For each query, sample a few elements and use binary search to check their frequencies.

efficiency: HIGH
time complexity: O(log n) per query
space complexity: O(n)
energy implications: LOW
explanation: The solution preprocesses the array to store each element's occurrences, allowing efficient frequency checks via binary search during queries. Sampling a few elements ensures quick determination of the majority element, leveraging the problem's constraint that the threshold exceeds half the subarray length.
