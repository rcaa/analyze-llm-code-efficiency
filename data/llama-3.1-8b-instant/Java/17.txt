ID: 2404
Title: Most Frequent Even Element
URL: https://leetcode.com/problems/most-frequent-even-element/
Status: OK

solution: 
```java
import java.util.HashMap;
import java.util.Map;

public class Main {
    public static int mostFrequentEven(int[] nums) {
        Map<Integer, Integer> countMap = new HashMap<>();
        int maxCount = 0;
        int result = -1;
        
        for (int num : nums) {
            if (num % 2 == 0) {
                countMap.put(num, countMap.getOrDefault(num, 0) + 1);
                if (countMap.get(num) > maxCount) {
                    maxCount = countMap.get(num);
                    result = num;
                } else if (countMap.get(num) == maxCount && num < result) {
                    result = num;
                }
            }
        }
        
        return result;
    }

    public static void main(String[] args) {
        int[] nums1 = {0,1,2,2,4,4,1};
        int[] nums2 = {4,4,4,9,2,4};
        int[] nums3 = {29,47,21,41,13,37,25,7};
        
        System.out.println(mostFrequentEven(nums1));  // Output: 2
        System.out.println(mostFrequentEven(nums2));  // Output: 4
        System.out.println(mostFrequentEven(nums3));  // Output: -1
    }
}
```

efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: This solution uses a HashMap to count the frequency of each even element in the array. It iterates through the array once, keeping track of the maximum frequency and the smallest even element with that frequency. This approach has a high efficiency due to its linear time complexity and low space complexity.
