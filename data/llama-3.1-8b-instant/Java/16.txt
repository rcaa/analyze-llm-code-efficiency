ID: 1984
Title: Minimum Difference Between Highest and Lowest of K Scores
URL: https://leetcode.com/problems/minimum-difference-between-highest-and-lowest-of-k-scores/
Status: OK

solution: 
```java
import java.util.Arrays;

public class Main {
    public static int minimumDifference(int[] nums, int k) {
        Arrays.sort(nums);
        int minDiff = Integer.MAX_VALUE;
        for (int i = 0; i <= nums.length - k; i++) {
            minDiff = Math.min(minDiff, nums[i + k - 1] - nums[i]);
        }
        return minDiff;
    }

    public static void main(String[] args) {
        int[] nums = {9, 4, 1, 7};
        int k = 2;
        System.out.println(minimumDifference(nums, k));
    }
}
```

efficiency: HIGH
time complexity: O(n log n)
space complexity: O(1)
energy implications: LOW
explanation: This solution sorts the array first, then iterates over the array to find the minimum difference between the highest and lowest score of k consecutive students. The time complexity is O(n log n) due to the sorting operation, and the space complexity is O(1) as it only uses a constant amount of space. The energy implications are low as the algorithm only performs a single sorting operation and a few iterations over the array.
