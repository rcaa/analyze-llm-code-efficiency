ID: 2441
Title: Largest Positive Integer That Exists With Its Negative
URL: https://leetcode.com/problems/largest-positive-integer-that-exists-with-its-negative/
Status: OK

solution: ```java
public int largestSumOfAverages(int[] nums) {
    int n = nums.length;
    int[] prefixSum = new int[n + 1];
    for (int i = 0; i < n; i++) {
        prefixSum[i + 1] = prefixSum[i] + nums[i];
    }
    double[][] dp = new double[n][n];
    for (int i = 0; i < n; i++) {
        dp[i][i] = (double) nums[i];
    }
    for (int len = 2; len <= n; len++) {
        for (int i = 0; i <= n - len; i++) {
            int j = i + len - 1;
            dp[i][j] = -1;
            for (int k = i; k < j; k++) {
                if (dp[i][k] != -1 && dp[k + 1][j] != -1) {
                    dp[i][j] = Math.max(dp[i][j], dp[i][k] + dp[k + 1][j]);
                }
            }
        }
    }
    return (int) dp[0][n - 1];
}
```
efficiency: HIGH
time complexity: O(n^3)
space complexity: O(n^2)
energy implications: MEDIUM
explanation: This solution uses dynamic programming to find the largest sum of averages. It first calculates the prefix sum of the array, then uses a 2D dp array to store the maximum sum of averages for each subarray. The dp array is filled in a bottom-up manner, and the final result is stored in dp[0][n-1]. The time complexity is O(n^3) due to the nested loops, and the space complexity is O(n^2) for the dp array.
