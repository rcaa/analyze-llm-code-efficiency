ID: 2432
Title: The Employee That Worked on the Longest Task
URL: https://leetcode.com/problems/the-employee-that-worked-on-the-longest-task/
Status: OK

solution: ```java
public int hardestWorker(int n, int[][] logs) {
    int maxTime = 0;
    int maxId = 0;
    int prevTime = 0;
    for (int[] log : logs) {
        int id = log[0];
        int time = log[1];
        if (time - prevTime > maxTime) {
            maxTime = time - prevTime;
            maxId = id;
        } else if (time - prevTime == maxTime && id < maxId) {
            maxId = id;
        }
        prevTime = time;
    }
    return maxId;
}
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(1)
energy implications: LOW
explanation: This solution iterates through the logs array once, keeping track of the maximum time difference and the corresponding employee ID. It uses a single pass through the array, making it efficient in terms of time complexity. The space complexity is O(1) as it only uses a constant amount of space to store the maximum time difference and the corresponding employee ID.
