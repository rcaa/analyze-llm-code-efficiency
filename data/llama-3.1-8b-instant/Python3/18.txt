Title: Sort the People
URL: https://leetcode.com/problems/sort-the-people/
Status: OK

solution: 
```python
def sort_people(names, heights):
    return [name for _, name in sorted(zip(heights, names), reverse=True)]
```
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(n)
energy implications: LOW
explanation: This solution uses the built-in `sorted` function in Python, which uses the Timsort algorithm with a time complexity of O(n log n). It sorts the list of tuples (height, name) in descending order by height, and then uses a list comprehension to extract the names in the sorted order. The space complexity is O(n) because we need to store the sorted list of tuples.
