ID: 2404
Title: Most Frequent Even Element
URL: https://leetcode.com/problems/most-frequent-even-element/
Status: OK

solution: ```python
from collections import Counter

def most_frequent_even(nums):
    even_nums = [num for num in nums if num % 2 == 0]
    if not even_nums:
        return -1
    count = Counter(even_nums)
    max_count = max(count.values())
    most_frequent = [num for num, freq in count.items() if freq == max_count]
    return min(most_frequent)
```

efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: This solution uses a list comprehension to filter out the even numbers, then uses the Counter class from the collections module to count the frequency of each even number. It finds the maximum frequency and then uses another list comprehension to find all numbers with that frequency. Finally, it returns the smallest of these numbers. The time complexity is O(n) because we need to iterate over the list twice, and the space complexity is also O(n) because in the worst case, we need to store all numbers in the list.
