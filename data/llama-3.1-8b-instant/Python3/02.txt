Title: Number of Valid Words for Each Puzzle
URL: https://leetcode.com/problems/number-of-valid-words-for-each-puzzle/
Status: OK

solution: 
```python
def findNumOfValidWords(words, puzzles):
    word_count = {}
    for word in words:
        bit_mask = 0
        for char in word:
            bit_mask |= 1 << (ord(char) - ord('a'))
        word_count[bit_mask] = word_count.get(bit_mask, 0) + 1

    result = []
    for puzzle in puzzles:
        bit_mask = 0
        for char in puzzle:
            bit_mask |= 1 << (ord(char) - ord('a'))
        count = 0
        for i in range(1, 26):
            if (bit_mask & (1 << i)) and (word_count.get(bit_mask | (1 << i), 0)):
                count += word_count[bit_mask | (1 << i)]
        result.append(count)

    return result
```
efficiency: HIGH
time complexity: O(n * m * log(m))
space complexity: O(n)
energy implications: LOW
explanation: This solution uses a bit mask to represent each word and puzzle, allowing for efficient lookups and bitwise operations. It iterates over each puzzle, and for each puzzle, it iterates over each possible bit mask that can be formed by adding a single letter to the puzzle. If the resulting bit mask is in the word count dictionary, it increments the count. The result is an array of counts, where each count is the number of words in the given word list that is valid with respect to the puzzle.
