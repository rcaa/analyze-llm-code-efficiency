ID: 2432
Title: The Employee That Worked on the Longest Task
URL: https://leetcode.com/problems/the-employee-that-worked-on-the-longest-task/
Status: OK

solution: ```cpp
int longestWPI(vector<int>& logs) {
    int score = 0, res = 0;
    unordered_map<int, int> mp;
    for (int i = 0; i < logs.size(); i++) {
        if (logs[i][1] > logs[i][0]) score++;
        else score--;
        if (score > 0) res = i + 1;
        else if (mp.find(score) == mp.end()) mp[score] = i + 1;
    }
    return res;
}
```

efficiency: HIGH

time complexity: O(n)

space complexity: O(n)

energy implications: LOW

explanation: This solution uses a hash map to store the scores and their corresponding indices. It iterates through the logs array, updating the score and checking if the current score is greater than 0 or if the current score is in the hash map. If the current score is greater than 0, it updates the result. If the current score is not in the hash map, it adds the current score and its index to the hash map. The time complexity is O(n) because we are iterating through the logs array once, and the space complexity is O(n) because in the worst case, we need to store all the scores in the hash map.
