Title: Sort the People
URL: https://leetcode.com/problems/sort-the-people/
Status: OK

solution: `std::sort(names.begin(), names.end(), [&](const std::string& a, const std::string& b) { return heights[names.size() - 1 - std::distance(names.begin(), a)] > heights[names.size() - 1 - std::distance(names.begin(), b)]; });`
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(1)
energy implications: LOW
explanation: This solution uses the built-in `std::sort` function in C++ and a custom comparison function to sort the names in descending order by height. The comparison function calculates the height of each person by subtracting their index from the length of the names array, which is done in reverse order to achieve the desired sorting. This approach has a time complexity of O(n log n) due to the sorting operation, and a space complexity of O(1) since it only uses a constant amount of extra memory.
