ID: 2404
Title: Most Frequent Even Element
URL: https://leetcode.com/problems/most-frequent-even-element/
Status: OK

solution: 
```cpp
#include <unordered_map>
#include <vector>
#include <algorithm>

int mostFrequentEven(std::vector<int>& nums) {
    std::unordered_map<int, int> count;
    for (int num : nums) {
        if (num % 2 == 0) {
            count[num]++;
        }
    }
    int maxCount = 0;
    int result = -1;
    for (auto& pair : count) {
        if (pair.second > maxCount) {
            maxCount = pair.second;
            result = pair.first;
        } else if (pair.second == maxCount && pair.first < result) {
            result = pair.first;
        }
    }
    return result;
}
```

efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: This solution uses a hash map to count the frequency of each even element in the array. It then iterates through the hash map to find the most frequent even element. If there is a tie, it returns the smallest one. The time complexity is O(n) because we need to iterate through the array and the hash map. The space complexity is also O(n) because in the worst case, all elements in the array are stored in the hash map.
