ID: 2441
Title: Largest Positive Integer That Exists With Its Negative
URL: https://leetcode.com/problems/largest-positive-integer-that-exists-with-its-negative/
Status: OK

solution: 
```cpp
int largestVal(vector<int>& nums) {
    unordered_map<int, int> num_count;
    for (int num : nums) {
        num_count[num]++;
    }
    int max_k = -1;
    for (auto& pair : num_count) {
        if (pair.second > 1 && -pair.first >= 0 && num_count.count(-pair.first)) {
            max_k = max(max_k, pair.first);
        }
    }
    return max_k;
}
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: This solution uses an unordered map to count the occurrences of each number in the array. It then iterates over the map to find the largest positive integer k such that -k also exists in the array. The time complexity is O(n) because we are iterating over the array once to count the numbers and once to find the largest k. The space complexity is also O(n) because in the worst case, we need to store all numbers in the map.
