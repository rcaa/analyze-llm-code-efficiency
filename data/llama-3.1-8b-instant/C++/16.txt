ID: 1984
Title: Minimum Difference Between Highest and Lowest of K Scores
URL: https://leetcode.com/problems/minimum-difference-between-highest-and-lowest-of-k-scores/
Status: OK

solution: ```cpp
int minimumDifference(vector<int>& nums, int k) {
    sort(nums.begin(), nums.end());
    return nums[k-1] - nums[0];
}
```
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(1)
energy implications: LOW
explanation: This solution sorts the array in ascending order and then returns the difference between the kth smallest and the smallest element. This approach ensures that the difference between the highest and lowest of the k scores is minimized. The time complexity is O(n log n) due to the sorting operation, and the space complexity is O(1) as it only uses a constant amount of space.
