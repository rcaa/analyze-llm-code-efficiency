ID: 2432
Title: The Employee That Worked on the Longest Task
URL: https://leetcode.com/problems/the-employee-that-worked-on-the-longest-task/
Status: OK

solution: 
```cpp
#include <vector>
#include <algorithm>
#include <map>

using namespace std;

int longestTask(int n, vector<vector<int>>& logs) {
    map<int, int> employeeTimes;
    for (auto& log : logs) {
        employeeTimes[log[0]] += log[1];
    }
    int longestTimeEmployee = 0;
    int maxTime = 0;
    for (auto& [employee, time] : employeeTimes) {
        if (time > maxTime) {
            longestTimeEmployee = employee;
            maxTime = time;
        }
    }
    return longestTimeEmployee;
}
```
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(n)
energy implications: LOW
explanation: The solution utilizes a map to efficiently store the total time each employee spends working. It iterates through the logs, updating the employee's total time. Finally, it iterates through the map to find the employee with the maximum time, returning their ID. The time complexity is dominated by the map insertion and lookup operations, which have an average complexity of O(log n). The space complexity is O(n) due to the map storing employee times.
