ID: 1961
Title: Check If String Is a Prefix of Array
URL: https://leetcode.com/problems/check-if-string-is-a-prefix-of-array/
Status: OK

solution: 
```cpp
#include <string>
#include <vector>

bool isPrefixString(std::string s, std::vector<std::string>& words) {
    std::string prefix = "";
    for (std::string word : words) {
        prefix += word;
        if (prefix == s) {
            return true;
        }
    }
    return false;
}
```
efficiency: HIGH
time complexity: O(N)
space complexity: O(N)
energy implications: LOW
explanation: The solution iterates through the words array, concatenating them into a prefix string. It compares the prefix string with the target string s at each step. If a match is found, it returns true. Otherwise, it continues iterating until all words have been checked.
