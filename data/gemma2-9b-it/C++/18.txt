ID: 2418
Title: Sort the People
URL: https://leetcode.com/problems/sort-the-people/
Status: OK

solution: 
```cpp
#include <vector>
#include <algorithm>

std::vector<std::string> sortPeople(std::vector<std::string>& names, std::vector<int>& heights) {
    std::vector<std::pair<int, std::string>> pairs(names.size());
    for (int i = 0; i < names.size(); ++i) {
        pairs[i] = {heights[i], names[i]};
    }
    std::sort(pairs.rbegin(), pairs.rend());
    std::vector<std::string> sortedNames;
    for (auto& pair : pairs) {
        sortedNames.push_back(pair.second);
    }
    return sortedNames;
}
```
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(n)
energy implications: LOW
explanation: The solution sorts the names based on heights using a custom comparator in `std::sort`. It creates pairs of height and name, sorts them in descending order of height, and extracts the names to return the sorted list.
