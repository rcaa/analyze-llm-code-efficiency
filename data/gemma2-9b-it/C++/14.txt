ID: 1929
Title: Concatenation of Array
URL: https://leetcode.com/problems/concatenation-of-array/
Status: OK

solution: 
```cpp
#include <vector>

std::vector<int> extendArray(std::vector<int>& nums) {
    std::vector<int> ans(nums.size() * 2);
    for (int i = 0; i < nums.size(); ++i) {
        ans[i] = nums[i];
        ans[i + nums.size()] = nums[i];
    }
    return ans;
}
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: The solution creates a new vector `ans` twice the size of the input vector `nums`. It then iterates through `nums`, copying each element into `ans` at its original index and then again at an index `n` positions ahead. This effectively concatenates two copies of `nums`.
