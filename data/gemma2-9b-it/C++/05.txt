ID: 3045
Title: Count Prefix and Suffix Pairs II
URL: https://leetcode.com/problems/count-prefix-and-suffix-pairs-ii/
Status: OK

solution: 
```cpp
#include <iostream>
#include <vector>
#include <string>

using namespace std;

bool isPrefixAndSuffix(string str1, string str2) {
    return str2.find(str1) == 0 && str2.find(str1) == str2.length() - str1.length();
}

int countPairs(vector<string>& words) {
    int count = 0;
    for (int i = 0; i < words.size(); i++) {
        for (int j = i + 1; j < words.size(); j++) {
            if (isPrefixAndSuffix(words[i], words[j])) {
                count++;
            }
        }
    }
    return count;
}

int main() {
    vector<string> words = {"a","aba","ababa","aa"};
    int result = countPairs(words);
    cout << result << endl;
    return 0;
}
```
efficiency: MEDIUM
time complexity: O(n^2)
space complexity: O(1)
energy implications: LOW
explanation: The solution uses a nested loop to iterate through all possible pairs of words. For each pair, it calls a helper function `isPrefixAndSuffix` to check if the first word is both a prefix and suffix of the second word. The count of valid pairs is then returned.  The time complexity is O(n^2) due to the nested loops, and the space complexity is O(1) as the algorithm uses a constant amount of extra space.
