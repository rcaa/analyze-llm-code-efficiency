ID: 1984
Title: Minimum Difference Between Highest and Lowest of K Scores
URL: https://leetcode.com/problems/minimum-difference-between-highest-and-lowest-of-k-scores/
Status: OK

solution: 
```cpp
#include <algorithm>
#include <vector>

int minimizeDifference(std::vector<int>& nums, int k) {
    std::sort(nums.begin(), nums.end());
    int minDiff = nums[k - 1] - nums[0];
    for (int i = 1; i <= nums.size() - k; ++i) {
        minDiff = std::min(minDiff, nums[i + k - 1] - nums[i]);
    }
    return minDiff;
}
```
efficiency: HIGH
time complexity: O(N log N)
space complexity: O(1)
energy implications: LOW
explanation: Sorts the array and iterates through possible k-sized subarrays, calculating the difference between the highest and lowest score in each subarray. The minimum difference found is returned.  Sorting dominates the time complexity.
