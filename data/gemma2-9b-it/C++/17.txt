ID: 2404
Title: Most Frequent Even Element
URL: https://leetcode.com/problems/most-frequent-even-element/
Status: OK

solution: 
```cpp
#include <vector>
#include <unordered_map>
#include <algorithm>

int mostFrequentEven(std::vector<int>& nums) {
    std::unordered_map<int, int> frequency;
    for (int num : nums) {
        if (num % 2 == 0) {
            frequency[num]++;
        }
    }
    int maxFrequency = 0;
    int result = -1;
    for (auto it : frequency) {
        if (it.second > maxFrequency) {
            maxFrequency = it.second;
            result = it.first;
        } else if (it.second == maxFrequency && it.first < result) {
            result = it.first;
        }
    }
    return result;
}
```
efficiency: HIGH
time complexity: O(N)
space complexity: O(N)
energy implications: LOW
explanation: The solution uses a hash map to store the frequency of each even number. It then iterates through the hash map to find the even number with the highest frequency and the smallest value in case of a tie.
