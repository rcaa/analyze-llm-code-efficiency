ID: 1799
Title: Maximize Score After N Operations
URL: https://leetcode.com/problems/maximize-score-after-n-operations/
Status: OK

solution: 
```cpp
#include <vector>
#include <algorithm>

using namespace std;

int maxScore(vector<int>& nums) {
    int n = nums.size() / 2;
    sort(nums.begin(), nums.end());
    int score = 0;
    for (int i = 0; i < n; ++i) {
        score += (i + 1) * __gcd(nums[i], nums[n + i]);
    }
    return score;
}

int gcd(int a, int b) {
    if (b == 0) {
        return a;
    }
    return gcd(b, a % b);
}
```
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(1)
energy implications: LOW
explanation: The solution sorts the array to efficiently pair elements with the highest possible GCD for each operation, maximizing the score.  The GCD is calculated using a recursive function.
