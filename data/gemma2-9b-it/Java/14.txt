ID: 1929
Title: Concatenation of Array
URL: https://leetcode.com/problems/concatenation-of-array/
Status: OK

solution: 
```java
int[] ans = new int[2*nums.length];
for (int i = 0; i < nums.length; i++) {
    ans[i] = nums[i];
    ans[i + nums.length] = nums[i];
}
return ans;
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: The solution iterates through the input array once, copying each element to its corresponding position in the output array. This results in a linear time complexity and space complexity.
