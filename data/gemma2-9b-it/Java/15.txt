ID: 1961
Title: Check If String Is a Prefix of Array
URL: https://leetcode.com/problems/check-if-string-is-a-prefix-of-array/
Status: OK

solution: 
```java
public class Solution {
    public boolean isPrefixString(String s, String[] words) {
        StringBuilder prefix = new StringBuilder();
        for (String word : words) {
            prefix.append(word);
            if (prefix.toString().equals(s)) {
                return true;
            }
        }
        return false;
    }
}
```
efficiency: HIGH
time complexity: O(N)
space complexity: O(N)
energy implications: LOW
explanation: The solution iterates through the words array, concatenating each word to a StringBuilder. It compares the concatenated string to the target string 's'. If a match is found, it returns true. Otherwise, it continues iterating until all words are checked or a match is found.
