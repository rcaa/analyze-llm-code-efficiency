Title: Online Majority Element In Subarray
URL: https://leetcode.com/problems/online-majority-element-in-subarray/
Status: OK

solution: 
```java
class MajorityChecker {
    private int[] arr;
    private int n;

    public MajorityChecker(int[] arr) {
        this.arr = arr;
        this.n = arr.length;
    }

    public int query(int left, int right, int threshold) {
        Map<Integer, Integer> counts = new HashMap<>();
        for (int i = left; i <= right; i++) {
            counts.put(arr[i], counts.getOrDefault(arr[i], 0) + 1);
        }
        for (int num : counts.keySet()) {
            if (counts.get(num) >= threshold) {
                return num;
            }
        }
        return -1;
    }
}
```
efficiency: HIGH
time complexity: O(N)
space complexity: O(N)
energy implications: LOW
explanation: The solution utilizes a HashMap to store the frequency of each element within the subarray. It iterates through the subarray, populating the HashMap with element counts. Subsequently, it iterates through the HashMap, returning the first element with a count greater than or equal to the threshold. If no such element is found, it returns -1. This approach leverages the efficiency of HashMap lookups for fast retrieval of element counts.
