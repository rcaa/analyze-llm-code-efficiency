ID: 3045
Title: Count Prefix and Suffix Pairs II
URL: https://leetcode.com/problems/count-prefix-and-suffix-pairs-ii/
Status: OK

solution: 
```java
int count = 0;
for (int i = 0; i < words.length - 1; i++) {
    for (int j = i + 1; j < words.length; j++) {
        if (words[i].length() > 0 && words[j].length() > 0 && words[i].equals(words[j].substring(0, words[i].length())) && words[j].endsWith(words[i])) {
            count++;
        }
    }
}
return count;
```
efficiency: HIGH
time complexity: O(N^2)
space complexity: O(1)
energy implications: LOW
explanation: The solution uses nested loops to iterate through all possible pairs of words. For each pair, it checks if the first word is both a prefix and a suffix of the second word. This approach has a time complexity of O(N^2) due to the nested loops, where N is the length of the input array. The space complexity is O(1) as it uses a constant amount of extra space.  The energy implications are low as the algorithm performs a limited number of operations.
