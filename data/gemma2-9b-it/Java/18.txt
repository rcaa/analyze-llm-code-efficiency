ID: 2418
Title: Sort the People
URL: https://leetcode.com/problems/sort-the-people/
Status: OK

solution: 
```java
import java.util.Arrays;
import java.util.Comparator;

class Solution {
    public String[] sortPeople(String[] names, int[] heights) {
        Pair[] pairs = new Pair[names.length];
        for (int i = 0; i < names.length; i++) {
            pairs[i] = new Pair(names[i], heights[i]);
        }
        Arrays.sort(pairs, Comparator.comparingInt(a -> a.height).reversed());
        String[] result = new String[names.length];
        for (int i = 0; i < names.length; i++) {
            result[i] = pairs[i].name;
        }
        return result;
    }

    class Pair {
        String name;
        int height;

        public Pair(String name, int height) {
            this.name = name;
            this.height = height;
        }
    }
}
```
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(n)
energy implications: LOW
explanation: The solution utilizes a custom Pair class to store name and height information. It then sorts an array of these pairs based on height in descending order using Arrays.sort and a custom comparator. Finally, it extracts the names from the sorted pairs to produce the desired output. This approach leverages the efficiency of the built-in sorting algorithm and maintains a reasonable space complexity.
