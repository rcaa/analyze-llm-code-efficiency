ID: 1799
Title: Maximize Score After N Operations
URL: https://leetcode.com/problems/maximize-score-after-n-operations/
Status: OK

solution: 
```java
import java.util.Arrays;

class Solution {
    public int maxScore(int[] nums) {
        Arrays.sort(nums);
        int score = 0;
        for (int i = 0; i < nums.length / 2; i++) {
            score += (i + 1) * gcd(nums[i], nums[nums.length - i - 1]);
        }
        return score;
    }
    private int gcd(int a, int b) {
        if (b == 0) {
            return a;
        }
        return gcd(b, a % b);
    }
}
```
efficiency: HIGH
time complexity: O(N log N)
space complexity: O(1)
energy implications: LOW
explanation: The solution sorts the array and then iterates through half of it, calculating the GCD of the smallest and largest elements in each iteration. This approach leverages the fact that the GCD of two numbers is maximized when they are as close as possible in value. Sorting the array ensures that the smallest and largest elements are paired together, leading to the highest possible score.
