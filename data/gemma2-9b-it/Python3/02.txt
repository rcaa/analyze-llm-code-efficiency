Title: Number of Valid Words for Each Puzzle
URL: https://leetcode.com/problems/number-of-valid-words-for-each-puzzle/
Status: OK

solution: 
```python
def count_valid_words(words, puzzles):
    answer = []
    for puzzle in puzzles:
        count = 0
        for word in words:
            if word[0] == puzzle[0] and all(letter in puzzle for letter in word):
                count += 1
        answer.append(count)
    return answer
```
efficiency: HIGH
time complexity: O(N*M)
space complexity: O(1)
energy implications: LOW
explanation: The solution iterates through each puzzle and each word in the list. For each word, it checks if the word starts with the first letter of the puzzle and if all letters in the word are present in the puzzle. If both conditions are met, the word is counted as valid.  This results in a time complexity of O(N*M) where N is the number of puzzles and M is the number of words. The space complexity is O(1) as the algorithm uses a constant amount of extra space.
