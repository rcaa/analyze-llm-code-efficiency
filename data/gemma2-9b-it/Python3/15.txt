ID: 1961
Title: Check If String Is a Prefix of Array
URL: https://leetcode.com/problems/check-if-string-is-a-prefix-of-array/
Status: OK

solution: 
```python
def isPrefixString(s, words):
    prefix = ""
    for word in words:
        prefix += word
        if prefix == s:
            return True
    return False
```
efficiency: HIGH
time complexity: O(N)
space complexity: O(N)
energy implications: LOW
explanation: The solution iterates through the words array, concatenating each word to a prefix string. It compares the prefix string to the target string s. If a match is found, it returns True. Otherwise, it continues iterating until all words have been checked.  The time complexity is linear as it iterates through the words array once. The space complexity is also linear due to the potential size of the prefix string.
