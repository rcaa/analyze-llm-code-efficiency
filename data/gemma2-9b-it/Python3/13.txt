ID: 1207
Title: Unique Number of Occurrences
URL: https://leetcode.com/problems/unique-number-of-occurrences/
Status: OK

solution: 
```python
from collections import Counter
def uniqueOccurrences(arr):
    counts = Counter(arr)
    return len(set(counts.values())) == len(counts)
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: The solution uses a Counter to efficiently count the occurrences of each element in the array. Then, it converts the values (counts) into a set to identify unique occurrences. Finally, it compares the size of the set with the size of the Counter to determine if all counts are unique.  This approach leverages the efficiency of built-in Python data structures.
