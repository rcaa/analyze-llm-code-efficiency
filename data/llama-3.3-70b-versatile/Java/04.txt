Title: Maximize Score After N Operations
URL: https://leetcode.com/problems/maximize-score-after-n-operations/
Status: OK

solution:
import java.util.*;

public class Main {
    public static int maxScore(int[] nums) {
        int n = nums.length / 2;
        int[] memo = new int[1 << n];
        Arrays.fill(memo, -1);
        return dfs(nums, 0, n, memo);
    }

    public static int dfs(int[] nums, int mask, int n, int[] memo) {
        if (Integer.bitCount(mask) == n) {
            return 0;
        }
        if (memo[mask] != -1) {
            return memo[mask];
        }
        int res = 0;
        for (int i = 0; i < 2 * n; i++) {
            for (int j = i + 1; j < 2 * n; j++) {
                if (((mask >> i) & 1) == 0 && ((mask >> j) & 1) == 0) {
                    int newMask = mask | (1 << i) | (1 << j);
                    int score = Integer.bitCount(mask) / 2 + 1;
                    score *= gcd(nums[i], nums[j]);
                    res = Math.max(res, score + dfs(nums, newMask, n, memo));
                }
            }
        }
        memo[mask] = res;
        return res;
    }

    public static int gcd(int a, int b) {
        if (b == 0) {
            return a;
        }
        return gcd(b, a % b);
    }
}

efficiency: MEDIUM
time complexity: O(2^n * n^2)
space complexity: O(2^n)
energy implications: MEDIUM
explanation: The solution uses depth-first search with memoization to explore all possible operations and calculate the maximum score. The key strength is the use of memoization to avoid redundant calculations, while the weakness is the exponential time complexity due to the recursive nature of the algorithm.
