ID: 3045
Title: Count Prefix and Suffix Pairs II
URL: https://leetcode.com/problems/count-prefix-and-suffix-pairs-ii/
Status: OK

solution:
public int countPrefixAndSuffixPairs(String[] words) {
    int count = 0;
    for (int i = 0; i < words.length; i++) {
        for (int j = i + 1; j < words.length; j++) {
            if (words[j].startsWith(words[i]) && words[j].endsWith(words[i])) {
                count++;
            }
        }
    }
    return count;
}

efficiency: MEDIUM
time complexity: O(n^2 * m)
space complexity: O(1)
energy implications: MEDIUM
explanation: The provided Java solution iterates through each pair of strings in the input array, checking if the first string is both a prefix and a suffix of the second string. This approach has a time complexity of O(n^2 * m) due to the nested loops and string operations, where n is the number of strings and m is the maximum length of a string. The space complexity is O(1) as it only uses a constant amount of space to store the count variable. The energy implications are medium due to the computational overhead of the nested loops and string operations.
