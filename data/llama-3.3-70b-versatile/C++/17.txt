Title: Most Frequent Even Element
URL: https://leetcode.com/problems/most-frequent-even-element/
Status: OK

solution:
unordered_map<int, int> count;
int max_count = 0, res = -1;
for (int num : nums) {
    if (num % 2 == 0) {
        count[num]++;
        if (count[num] > max_count) {
            max_count = count[num];
            res = num;
        } else if (count[num] == max_count && num < res) {
            res = num;
        }
    }
}
return res;

efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: This solution uses an unordered map to count the frequency of each even element in the array, keeping track of the most frequent one and updating it if a smaller one with the same frequency is found. If no even elements are found, it returns -1. The use of an unordered map allows for efficient lookups and updates, resulting in a high efficiency and low energy implications.
