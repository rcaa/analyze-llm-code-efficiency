ID: 1961
Title: Check If String Is a Prefix of Array
URL: https://leetcode.com/problems/check-if-string-is-a-prefix-of-array/
Status: OK

solution: 
```cpp
class Solution {
public:
    bool isPrefixString(string s, vector<string>& words) {
        string prefix = "";
        for (const auto& word : words) {
            prefix += word;
            if (prefix == s) {
                return true;
            }
        }
        return false;
    }
};
```
efficiency: HIGH
time complexity: O(n*m)
space complexity: O(n*m)
energy implications: LOW
explanation: A string is checked against the concatenation of the first k strings in the array, returning true if a match is found, and false otherwise.
