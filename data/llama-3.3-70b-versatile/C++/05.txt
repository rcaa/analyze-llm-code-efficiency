Title: Count Prefix and Suffix Pairs II
URL: https://leetcode.com/problems/count-prefix-and-suffix-pairs-ii/
Status: OK

solution:
int countPairs(vector<string>& words) {
    int count = 0;
    for (int i = 0; i < words.size(); i++) {
        for (int j = i + 1; j < words.size(); j++) {
            if (words[i].size() <= words[j].size() && 
                words[j].substr(0, words[i].size()) == words[i] && 
                words[j].substr(words[j].size() - words[i].size()) == words[i]) {
                count++;
            }
            if (words[j].size() <= words[i].size() && 
                words[i].substr(0, words[j].size()) == words[j] && 
                words[i].substr(words[i].size() - words[j].size()) == words[j]) {
                count++;
            }
        }
    }
    return count;
}
efficiency: MEDIUM
time complexity: O(n^2 * m)
space complexity: O(1)
energy implications: MEDIUM
explanation: The provided solution iterates over each pair of words in the input array and checks if the shorter word is both a prefix and a suffix of the longer word. This approach has a time complexity of O(n^2 * m), where n is the number of words and m is the maximum length of a word. The space complexity is O(1) as it only uses a constant amount of space to store the count of pairs. The energy implications are medium due to the nested loops and string operations.
