ID: 1037
Title: Valid Boomerang
URL: https://leetcode.com/problems/valid-boomerang/
Status: OK

solution: 
```cpp
class Solution {
public:
    bool isBoomerang(vector<vector<int>>& points) {
        return (points[1][1] - points[0][1]) * (points[2][0] - points[1][0]) != (points[2][1] - points[1][1]) * (points[1][0] - points[0][0]);
    }
};
```
efficiency: HIGH
time complexity: O(1)
space complexity: O(1)
energy implications: LOW
explanation: A boomerang is formed when three points are not collinear. This solution checks for collinearity by comparing the slopes between pairs of points, returning true if the points are not collinear and false otherwise.
