ID: 1984
Title: Minimum Difference Between Highest and Lowest of K Scores
URL: https://leetcode.com/problems/minimum-difference-between-highest-and-lowest-of-k-scores/
Status: OK

solution:
int minimumDifference(vector<int>& nums, int k) {
    sort(nums.begin(), nums.end());
    int min_diff = INT_MAX;
    for (int i = 0; i <= nums.size() - k; i++) {
        min_diff = min(min_diff, nums[i + k - 1] - nums[i]);
    }
    return min_diff;
}
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(1)
energy implications: LOW
explanation: The solution sorts the input array and then iterates over the array to find the minimum difference between the highest and lowest scores of k students. The time complexity is dominated by the sorting operation, and the space complexity is constant as no additional space is used. The energy implications are low as the solution only involves a single pass over the array after sorting.
