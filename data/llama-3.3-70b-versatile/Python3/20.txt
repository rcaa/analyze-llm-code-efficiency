Title: Largest Positive Integer That Exists With Its Negative
URL: https://leetcode.com/problems/largest-positive-integer-that-exists-with-its-negative/
Status: OK

solution: 
```
def find_k(nums):
    nums_set = set(nums)
    k = -1
    for num in nums:
        if num > 0 and -num in nums_set:
            k = max(k, num)
    return k
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: The solution uses a set to store the input array for efficient lookups, then iterates through the array to find the largest positive integer k with a corresponding negative value, returning -1 if no such integer exists.
