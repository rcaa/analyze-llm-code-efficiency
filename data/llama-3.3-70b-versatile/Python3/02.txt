Title: Number of Valid Words for Each Puzzle
URL: https://leetcode.com/problems/number-of-valid-words-for-each-puzzle/
Status: OK

solution: 
```python
def findNumOfValidWords(words, puzzles):
    def is_valid(word, puzzle):
        return puzzle[0] in word and all(c in puzzle for c in word)

    result = []
    for puzzle in puzzles:
        count = sum(is_valid(word, puzzle) for word in words)
        result.append(count)
    return result
```
efficiency: HIGH
time complexity: O(n*m*k)
space complexity: O(n)
energy implications: LOW
explanation: The solution uses a simple iterative approach to check each word against each puzzle, utilizing a helper function `is_valid` to verify the conditions for a valid word. This approach ensures efficiency by only iterating through the words and puzzles as necessary, minimizing computational overhead.
