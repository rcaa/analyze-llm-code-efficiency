ID: 1984
Title: Minimum Difference Between Highest and Lowest of K Scores
URL: https://leetcode.com/problems/minimum-difference-between-highest-and-lowest-of-k-scores/
Status: OK

solution: 
```python
def minimumDifference(nums, k):
    nums.sort()
    return min(nums[i+k-1] - nums[i] for i in range(len(nums) - k + 1))
```
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(n)
energy implications: LOW
explanation: The solution sorts the input array and then uses a sliding window approach to find the minimum difference between the highest and lowest scores of k students. The time complexity is dominated by the sorting step, and the space complexity is linear due to the sorting. The energy implications are low since the algorithm only uses a single pass through the sorted array.
