Title: Unique Number of Occurrences
URL: https://leetcode.com/problems/unique-number-of-occurrences/
Status: OK

solution: 
```python
def unique_occurrences(arr):
    count_dict = {}
    for num in arr:
        if num in count_dict:
            count_dict[num] += 1
        else:
            count_dict[num] = 1
    return len(count_dict.values()) == len(set(count_dict.values()))
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: Utiliza um dicionário para contar as ocorrências de cada valor no array e verifica se o número de ocorrências é único comparando o número de valores únicos no dicionário com o número de valores no dicionário.
