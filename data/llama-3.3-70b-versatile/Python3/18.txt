Title: Sort the People
URL: https://leetcode.com/problems/sort-the-people/
Status: OK

solution: ```python
def sort_people(names, heights):
    return [name for _, name in sorted(zip(heights, names), reverse=True)]
```
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(n)
energy implications: LOW
explanation: The solution uses Python's built-in sorting function with a custom key to sort the names based on the corresponding heights in descending order. The zip function is used to pair each name with its height, and the sorted function returns a new list of tuples, which is then unpacked to get the sorted list of names.
