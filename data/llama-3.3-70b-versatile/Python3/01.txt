Title: Online Majority Element In Subarray
URL: https://leetcode.com/problems/online-majority-element-in-subarray/
Status: OK

solution: 
```python
from collections import Counter

class MajorityChecker:
    def __init__(self, arr):
        self.arr = arr

    def query(self, left, right, threshold):
        subarray = self.arr[left:right+1]
        counter = Counter(subarray)
        for num, count in counter.items():
            if count >= threshold:
                return num
        return -1
```
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: The MajorityChecker class uses a Counter to count the occurrences of each element in the subarray, then returns the first element that meets the threshold. If no such element exists, it returns -1. This approach is efficient because it only requires a single pass through the subarray.
