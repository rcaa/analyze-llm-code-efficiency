Title: Minimum Difference Between Highest and Lowest of K Scores
URL: https://leetcode.com/problems/minimum-difference-between-highest-and-lowest-of-k-scores/
Status: OK

solution: public int minScore(int[] nums, int k) { Arrays.sort(nums); int minDiff = Integer.MAX_VALUE; for (int i = 0; i <= nums.length - k; i++) { int current = nums[i + k - 1] - nums[i]; if (current < minDiff) minDiff = current; } return minDiff; }
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(1)
energy implications: LOW
explanation: Sorting the array allows checking only consecutive k-element windows, ensuring the minimal maximum-minimum difference is found efficiently. The approach leverages sorting to reduce the problem to a linear scan, minimizing computational overhead.
