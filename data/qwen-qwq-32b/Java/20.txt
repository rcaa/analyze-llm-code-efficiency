Title: Largest Positive Integer That Exists With Its Negative
URL: https://leetcode.com/problems/largest-positive-integer-that-exists-with-its-negative/
Status: OK

solution: public int largestPositive(int[] nums) {
    Set<Integer> set = new HashSet<>();
    for (int num : nums) {
        set.add(num);
    }
    int maxK = -1;
    for (int num : nums) {
        if (num > 0 && set.contains(-num)) {
            maxK = Math.max(maxK, num);
        }
    }
    return maxK == -1 ? -1 : maxK;
}
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: Uses a hash set for O(1) lookups, iterating elements twice (O(n) time). Tracks maximum valid k by checking each positive number's negative counterpart. Efficient and handles all edge cases within constraints.
