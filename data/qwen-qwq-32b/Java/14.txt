ID: 1929
Title: Concatenation of Array
URL: https://leetcode.com/problems/concatenation-of-array/
Status: OK

solution: public int[] getConcatenation(int[] nums) { int n = nums.length; int[] ans = new int[2 * n]; System.arraycopy(nums, 0, ans, 0, n); System.arraycopy(nums, 0, ans, n, n); return ans; }
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: Uses System.arraycopy twice for optimized array copying, achieving O(n) time with minimal overhead. Direct memory operations reduce per-element processing, though requires O(n) space for the output array.
