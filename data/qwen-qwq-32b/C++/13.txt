Title: Unique Number of Occurrences
URL: https://leetcode.com/problems/unique-number-of-occurrences/
Status: OK

solution: 
bool checkUniqueOccurrences(vector<int>& arr) {
    unordered_map<int, int> counts;
    for (int num : arr) {
        counts[num]++;
    }
    unordered_set<int> freqSet;
    for (const auto& p : counts) {
        freqSet.insert(p.second);
    }
    return freqSet.size() == counts.size();
}
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: Uses a hash map to count occurrences, then checks if the set of counts has the same size as the original count map, ensuring all counts are unique. Efficient O(n) time and space.
