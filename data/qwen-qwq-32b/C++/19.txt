ID: 2432
Title: The Employee That Worked on the Longest Task
URL: https://leetcode.com/problems/the-employee-that-worked-on-the-longest-task/
Status: OK

solution: int hardestWorker(int n, vector<vector<int>>& logs) { int max_duration = logs[0][1]; int best_id = logs[0][0]; for (int i = 1; i < logs.size(); ++i) { int duration = logs[i][1] - logs[i-1][1]; if (duration > max_duration) { max_duration = duration; best_id = logs[i][0]; } else if (duration == max_duration) { if (logs[i][0] < best_id) best_id = logs[i][0]; } } return best_id; }
efficiency: HIGH
time complexity: O(L) where L is the length of logs
space complexity: O(1)
energy implications: LOW
explanation: The solution iterates through the logs once, calculating each task's duration and tracking the maximum duration and the smallest ID in case of ties. This approach ensures linear time and constant space, efficiently handling the problem constraints.
