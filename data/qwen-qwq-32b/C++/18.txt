ID: 2418
Title: Sort the People
URL: https://leetcode.com/problems/sort-the-people/
Status: OK

solution: vector<string> sortNamesByHeight(vector<string>& names, vector<int>& heights) {
    vector<pair<int, string>> people;
    for(int i = 0; i < names.size(); ++i) {
        people.emplace_back(heights[i], names[i]);
    }
    sort(people.rbegin(), people.rend(), [](const pair<int,string>& a, const pair<int,string>& b) { return a.first < b.first; });
    vector<string> result;
    for(const auto& p : people) {
        result.push_back(p.second);
    }
    return result;
}
efficiency: HIGH
time complexity: O(n log n)
space complexity: O(n)
energy implications: LOW
explanation: Pairs of heights and names are created, sorted in descending order by height using a comparator, then names are extracted. Efficient sorting with O(n log n) time and linear space.
