Title: Largest Positive Integer That Exists With Its Negative
URL: https://leetcode.com/problems/largest-positive-integer-that-exists-with-its-negative/
Status: OK

solution: 
int findLargestK(vector<int>& nums) {
    unordered_set<int> s(nums.begin(), nums.end());
    int max_k = -1;
    for (int num : s) {
        if (num > 0 && s.count(-num)) {
            max_k = max(max_k, num);
        }
    }
    return max_k;
}
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: Uses a hash set for O(1) lookups, iterating through elements once to find the maximum valid k by checking if the negative counterpart exists. Linear time and space with minimal overhead.
