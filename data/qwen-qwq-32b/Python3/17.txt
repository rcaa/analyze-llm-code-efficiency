Title: Most Frequent Even Element
URL: https://leetcode.com/problems/most-frequent-even-element/
Status: OK

solution: 
from collections import Counter

def most_frequent_even(nums):
    evens = [x for x in nums if x % 2 == 0]
    if not evens:
        return -1
    counts = Counter(evens)
    max_freq = max(counts.values())
    candidates = [num for num in counts if counts[num] == max_freq]
    return min(candidates)

efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: Filters even elements, uses a counter to track frequencies, identifies elements with maximum frequency, and selects the smallest among ties. Linear time and space complexity ensures efficiency for input sizes up to 2000.
