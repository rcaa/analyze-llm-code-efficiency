Title: Unique Number of Occurrences
URL: https://leetcode.com/problems/unique-number-of-occurrences/
Status: OK

solution: 
from collections import Counter
class Solution:
    def uniqueOccurrences(self, arr: list[int]) -> bool:
        counts = Counter(arr)
        freq = counts.values()
        return len(freq) == len(set(freq))
efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: Uses a Counter to track element frequencies, then checks if the set of frequencies has the same size as the list of frequencies, ensuring uniqueness. Efficient and straightforward with linear time and space complexity.
