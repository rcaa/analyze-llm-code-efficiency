Title: Largest Positive Integer That Exists With Its Negative
URL: https://leetcode.com/problems/largest-positive-integer-that-exists-with-its-negative/
Status: OK

solution: def findLargestK(nums):
    s = set(nums)
    max_k = -1
    for num in nums:
        if num > 0 and -num in s:
            if num > max_k:
                max_k = num
    return max_k if max_k != -1 else -1

efficiency: HIGH
time complexity: O(n)
space complexity: O(n)
energy implications: LOW
explanation: Uses a set for O(1) lookups to check existence of -k for each positive k. Iterates once through the array to track the maximum valid k. Efficiently handles all cases with minimal overhead.
